# MetricAggregation

Represents aggregation of metrics.

| Enums                            |                       |
| :------------------------------- | :-------------------- |
| `METRIC_AGGREGATION_UNSPECIFIED` | Unspecified operator. |
| `TOTAL`                          | SUM operator.         |
| `MINIMUM`                        | Minimum operator.     |
| `MAXIMUM`                        | Maximum operator.     |
| `COUNT`                          | Count operator.       |
