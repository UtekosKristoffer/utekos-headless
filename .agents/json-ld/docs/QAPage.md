# QAPage

A Schema.org Type

```text
Thing > CreativeWork > WebPage > QAPage
```

- Canonical URL: <https://schema.org/QAPage>
- [Check for open issues](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+QAPage)

A QAPage is a WebPage focussed on a specific Question and its Answer(s), e.g. in a question answering site or
documenting Frequently Asked Questions (FAQs).
