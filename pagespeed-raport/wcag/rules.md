# WCAG 2.0 Level A & AA Rules

This document catalogs axe-core accessibility rules grouped by WCAG conformance
level and rule status. The source rule metadata is preserved, while the document
structure is normalized for faster scanning, navigation, and reuse.

## Section Guide

| Section                     | Purpose                                                          |
| --------------------------- | ---------------------------------------------------------------- |
| WCAG 2.0 Level A & AA Rules | Core rules mapped to WCAG 2.0 A and AA success criteria          |
| WCAG 2.1 Level A & AA Rules | Additional rules introduced for WCAG 2.1                         |
| WCAG 2.2 Level A & AA Rules | Rules aligned with WCAG 2.2 guidance                             |
| Best Practices Rules        | Industry-standard guidance not strictly tied to WCAG conformance |
| WCAG 2.x Level AAA Rules    | Automated checks for AAA-level criteria                          |
| Experimental Rules          | Rules still under evaluation in axe-core                         |
| Deprecated Rules            | Rules scheduled for removal or already deprecated                |
