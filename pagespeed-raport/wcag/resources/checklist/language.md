# Language

This document summarizes language-related accessibility guidance and the related
WCAG criteria.

| Topic             | Technique                                                                                                                                                                                                                                                                                                                                                                                                                          | Testing Methodology                                                                                          | Requirement Level | WCAG Criterion                                                              |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ | ----------------- | --------------------------------------------------------------------------- |
| Page Language     | [Page Language](https://dequeuniversity.com/class/semantic-structure-wcag-2.2/language/primary-language): The primary language of the page MUST be identified accurately, using a standard language code on the `<html>` element, for example `<html lang="en">` or `<html lang="fr">`.                                                                                                                                            | [Testing methodology 3.1.1.a](https://dequeuniversity.com/class/dequeway/interpreting/wcag/a/3.1.1/3.1.1.a)  | Required          | [WCAG 3.1.1](https://www.w3.org/WAI/WCAG22/Understanding/language-of-page)  |
| Language of Parts | [Language of Parts](https://dequeuniversity.com/class/semantic-structure-wcag-2.2/language/parts): Inline language changes MUST be identified with a valid `lang` attribute.                                                                                                                                                                                                                                                       | [Testing methodology 3.1.2.a](https://dequeuniversity.com/class/dequeway/interpreting/wcag/aa/3.1.2/3.1.2.a) | Required          | [WCAG 3.1.2](https://www.w3.org/WAI/WCAG22/Understanding/language-of-parts) |
| Language Code     | [Two-Character Language Code](https://dequeuniversity.com/class/semantic-structure-wcag-2.2/language/language-codes): The language code SHOULD be designated with a standard two-character [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) code, for example `lang="en"`, to achieve maximum support across screen readers and browsers, although other codes such as `lang="en-us"` are technically allowable. | N/A                                                                                                          | Best Practice     | N/A                                                                         |
